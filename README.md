# bottle

#### 介绍
这个仓库是bottle网站微框架的完整架构，让bottle更加django化

#### 软件架构
此框架整合bottle框架，对接jinja2模板，session管理，sqlalchmey数据库管理，让bottle更加专业化。

#### 需要安装第三方库（或者更多）
- - bottle\n
- - bottle-sqlalchemy
- - bottle-session
- - bottle_redis
- - Jinja2
- - mysql-connector
- - SQLAlchemy
- - tornado
- - Twisted
- - virtualenv
- - virtualenvwrapper-win
- - mysql-connector



#### 使用说明

1.  python migrate.py
2.  python manage.py
